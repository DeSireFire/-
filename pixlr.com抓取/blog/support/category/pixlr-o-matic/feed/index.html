But many people may now know that this darkroom-themed Flash app is available for download, too. It will work on a Mac or PC, but you will need to install Adobe AIR first. This is the only one of our legacy web-based apps that we offer as a download.
How do I get Adobe AIR?
Adobe AIR is a free desktop app that serves as a “”wrapper”” for Flash-based apps that otherwise would run in a browser. You can download Adobe AIR from Adobe’s website here.
After I install Adobe AIR, then what?
Once you’ve download AIR, you will want to download Pixlr-o-matic from here. Then, simply run this Pixlr-o-matic file (installer) you downloaded.
